git clone <link>
git branch -> tells which branch are we currently
git branch -a -> shows all branches
git checkout -b <branchname> -> creating a new branch called <branchname>
git push origin <branchname> -> pushing changes into github(inside <branchname> branch)
git checkout <branchname> -> to switch between branches
git branch <branch> -> also creates a new branch
git branch -d <branch> -> to delete a branch

**
TO MERGE INTO MAIN BRANCH
-> go to the branch you want to merge into(typically "main" branch)

THEN USE
git merge <branch> -> branch you want to merge from
